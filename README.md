# Smart Blind Stick 

## Overview
An assistive device for visually impaired people that detects obstacles and ground hazards using ultrasonic sensors.

## Features
- Obstacle detection (front)
- Pit/stairs detection (downward sensor)
- Real-time buzzer alert system
- Multi-level warning system

## Tech Stack
Arduino UNO, Ultrasonic Sensors, Embedded C

## Working
The system uses ultrasonic sensors to measure distance and alerts the user via buzzer when obstacles are detected.

## How to Run
Upload the code to Arduino using Arduino IDE.
